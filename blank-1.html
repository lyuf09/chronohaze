无障碍支持说明

最近更新：2026 年 2 月 7 日

1. 目标

本站是个人作品站，持续参考 WCAG 2.2 AA 的核心原则进行改进，尽量让更多访客在不同设备和辅助技术下都能顺畅浏览。

2. 当前已做

  • 使用语义化标题结构与主要内容分区。
  • 提供可点击的文本导航与可见的页面层级。
  • 图片提供 `alt` 描述（在可行范围内）。
  • 颜色与文字对比度优先按可读性设计。

3. 已知限制

  • 部分历史音频作品暂未提供完整文字转录。
  • 第三方外链页面不受本站直接控制。

4. 反馈与协助

如果你在访问中遇到障碍（例如键盘操作、阅读顺序、文本对比、媒体可达性等），请发送邮件说明页面地址与问题细节，我会优先修复。

Accessibility Statement

Last updated: February 7, 2026

1. Goal

This website is a personal portfolio. It is continuously improved with reference to core WCAG 2.2 AA principles, so more visitors can browse smoothly across different devices and assistive technologies.

2. What is currently implemented

  • Semantic heading structure and clear main content sections.
  • Clickable text navigation and visible page hierarchy.
  • `alt` descriptions for images where feasible.
  • Text and color contrast designed with readability first.

3. Known limitations

  • Some legacy audio works do not yet include full transcripts.
  • External third-party pages are outside this site's direct control.

4. Feedback and assistance

If you encounter accessibility barriers (for example keyboard usage, reading order, text contrast, or media accessibility), please email the page URL and issue details. I will prioritize fixing it.
